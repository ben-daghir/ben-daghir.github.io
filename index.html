Machine Learning for Satellite Crop Classification

View on GitHub

Introduction

This page documents our semester project for Georgia Tech's Spring 2020 undergrad Introduction to Machine Learning. The goal of this project was to create a program capable of classifying satellite images of crops using seven different labels: maize, cassava, common bean, maize & common bean, maize & cassava, maize & soybean, cassava & common bean.

Data

The dataset we use comes from the Radiant Earth MLHub. The images are of a crop feild in Kenya throughout the growing season (June to November). Over four thousand images are provided all together, 3286 of which are provided for training and 1402 are provided for testing. Images are taken every month within the growing cycle, and the dataset divides provides the snapshots as twelve spectral bands possible for the Sentinel-2 satellite. The feilds in the image are only a few pixels wide.
